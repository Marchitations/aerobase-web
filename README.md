# Frontend

--Pré-requisitos

Antes de executar o projeto localmente, instale:
nodeJS: v24.13.0
npm versão 11.6.2
Angular versão 21.1.0

--Execução

Clonar o projeto usando git clone https://github.com/Marchitations/aerobase-web.git

Após seguir instruções do README.md do projeto "backend" e clonar o repositório supracitado, execute o comando npm install e após finalizar execute ng serve para subir o front-end.

Acesse a url http://localhost:4200/ para acessar a aplicação.
